Traditional Konkani Style Base

Features: Coconut + red chilies + garlic, kokum for tang, medium spicy, signature coastal aroma.

Best for: Surmai, Bangda, Pomfret, Prawns

Malvani Fish Curry Base (Spicy)

Features: Fiery red chilies, dry coconut (khobra), Malvani masala — strong aroma & spicy kick.

Best for: Prawns, Mackerel, Shark fish

Goan Xacuti Fish Curry Base

Features: Complex masala with poppy seeds, coconut, roasted spices; deep brown, rich & creamy.

Best for: Pomfret, Basa, Crab

Goan Recheado Fish Curry / Paste Base

Features: Tangy, chili-loaded paste with vinegar + Kashmiri chili — bold flavour for fry & curry both.

Best for: Bangda, Prawns

Saraswat / Gauda Fish Curry Base (Mild)

Features: Coconut milk + cumin — mildly spiced, light pinkish curry.

Best for: Pomfret, fresh small fish

Malabari Fish Curry Base (Kerala Coastal)

Features: Coconut milk, curry leaves, green chilies — slightly sweet coastal flavour.

Best for: Tilapia, Pearl spot, Pomfret

Tikhla Style Konkani Curry Base (Non-Coconut)

Features: Onion + tomato + red chilies, no coconut — gravy-like and spicy.

Best for: Mackerel, Surmai

Kolambi / Prawn Curry Base

Features: Designed especially for prawns — coconut + kokum + coriander, slightly sweet flavour.

Best for: Prawns only

Kombdi / Kalvan Style Fish Curry Base

Features: Ground roasted masala — bold red colour, coconut + red chili + garlic.

Best for: Surmai, river fish

Ghati Masala Fish Curry Base

Features: Dry-spice heavy, no coconut — smoky flavour, very spicy.

Best for: Mackerel, Tiger prawns

Coconut Milk (White Curry) Base

Features: Creamy & mild — coconut milk + ginger + green chili, soothing curry base.

Best for: Pomfret, Basa, Squid

Kokum Kadhi Style Curry Base (Light Pink Base)

Features: Kokum + coconut milk + mild spices — fresh pinkish broth, gentle flavour.

Best for: Delicate fish like Pomfret

Goan Ambotik Curry Base (Hot & Sour)

Features: Tamarind + red chili, no coconut — spicy and tangy curry base.

Best for: Shark fish, Mackerel

Konkan Dry Fish Curry Base

Features: Extra garlic, spicy & deep flavour — ideal for dry fish (Bombil / Sukkit) recipes.

Best for: Dry fish dishes
